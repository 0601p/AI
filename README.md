Transformer/ : transformer로 간단한 번역 task [<transformer 논문>](https://arxiv.org/abs/1706.03762)
VAE/ : VAE로 mnist 데이터 학습 [<VAE 논문>](https://arxiv.org/abs/1312.6114)
